ai-agent-helm-charts

Unofficial community Helm charts for deploying AI agents on Kubernetes.

Unofficial / not affiliated with Nous Research, Hermes Agent, OpenClaw, openclaw.rocks, or Paperclip Inc. Container images are pulled from their upstream registries under their own licenses.

Install

Classic Helm repository:

helm repo add ketaloca https://ketaloca.github.io/ai-agent-helm-charts
helm repo update
helm install my-hermes ketaloca/hermes-agent

…or OCI (no helm repo add needed):

helm install my-hermes oci://ghcr.io/ketaloca/charts/hermes-agent --version 0.1.1
