# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful, assume good faith, and keep discussion constructive and
on-topic. Harassment, discrimination, and personal attacks are not tolerated.

## Reporting

Report unacceptable behavior by opening a confidential report via the contact in
[SECURITY.md](SECURITY.md) (or the repository owner). Reports will be reviewed and
handled as promptly as is reasonable for a volunteer-maintained project.

The full text of the Contributor Covenant 2.1 is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
